# Derby-Invite

## Installation
npm install derby-invite
### Setting
#### Step 1. Require
```javascript
var derbyInvite = require('derby-invite');
```
#### Step 2. Options
```javascript
var options = {};
```

#### Step 3. Middleware
```javascript
  .use(derbyInvite.middleware(options))
```
